Frequently Asked Questions
How do I reset a user's password? −
Go to User Management → find the user → click Edit → scroll to Security section → click "Send Password Reset Email". The user will receive a secure link valid for 24 hours.
How do I configure JWT token expiry? +
In appsettings.json, set JwtSettings.AccessTokenExpiry (in minutes) and JwtSettings.RefreshTokenExpiry (in days). Defaults are 15 minutes and 7 days. Redeploy required after changes.
How do I generate a custom report? +
Navigate to Reports → New Report. Select a template (FastReport or RDLC), choose your data source and filters, then click Generate. Reports can be exported as PDF, Excel, or HTML.
How do I integrate with Azure DevOps pipelines? +
See the CI/CD guide in the documentation. You'll need to add the NexusAdmin deployment task to your azure-pipelines.yml and configure the API key in Azure Key Vault.
